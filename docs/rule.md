minecraftで行うチーム戦のPvPバトル

/kcdk game start

を押下すると
スタートアップコマンドが実行
チームごとに待機地点にTPされその場から動けなくなる
待機地点が設定されていない場合はリスポーン地点にそのまま移動する
待機地点が設定されていた場合は待機時間がカウントされて0になるとリスポーン地点に移動する

リスポーン地点に移動した後チームにエフェクトが設定されている場合はプレイヤーに付与される

その後プレイヤーが死亡するとリスポーン可能回数が消費されリスポーン地点に移動する
リスポーン可能回数が-1の場合は無限にリスポーンできる
リスポーン回数が0の状態で死ぬとプレイヤーはスペクテイターとなりゲームから死亡したものとみなす

設定された制限時間を満たす
または終了条件を満たすとゲームが終了
設定された終了条件に設定されたメッセージがタイトルとして全員に表示される

終了後
シャットダウンコマンドが実行されてゲーム終了

/kcdk game stop

はゲームの強制終了コマンド

備考
UIについて
ゲームの状況はアクションバーを使って表示する
制限時間(設定がなければ経過時間) チームごとの生存人数　が表示される

プレイヤーのkill deathの値を保持しておく
プレイヤーがkillしたときタイトルで0.5秒間
title: n Kill
subtitle: killしたプレイヤー名
を表示する

ゲーム終了後プレイヤーのtab名をいかに変更する
mcid k:キル数/d:デス数
